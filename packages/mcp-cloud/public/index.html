Design Mode MCP

This is the hosted MCP relay for the Design Mode Chrome extension. Coding agents call its tools to read and edit whatever page you have open in the browser — no terminal, no local server.

Tools

  • get_changes — every style, text, DOM, and comment edit in the current session.
  • apply_changes — push CSS back to the page for live preview.
  • clear_changes — reset the session.
  • get_session_summary — connection status + counts.
  • export_changes — emit edits as css, tailwind, scss, or jsx.
  • get_screenshot — PNG of the page or one cropped element.

Limits

  • 25 tool calls per device per UTC day.
  • The Design Mode side panel must stay open for calls to land.
  • Anonymous device tokens, no account, one-click revoke.

Endpoint: POST /api/mcp with a Bearer dm_… token. Generate one from the side panel's MCP settings, or by POSTing to /api/auth/register.
