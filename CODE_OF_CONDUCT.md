# Code of Conduct

This project adopts the [Contributor Covenant, version 2.1][covenant] as its
code of conduct. By participating in this project — through issues, pull
requests, discussions, or any other channel — you agree to abide by it.

The full text is maintained upstream and applies to all project spaces:
https://www.contributor-covenant.org/version/2/1/code_of_conduct/

## In short

- Be respectful. Disagreement is welcome; personal attacks are not.
- Assume good intent; ask before assuming bad.
- Critique the work, not the person.
- Help newcomers — most of us were new once.

## Reporting

If you experience or witness behavior that goes against the covenant, contact
the maintainer directly:

**Sandeep Baskaran** — `hello@sandeepbaskaran.com`

Reports are kept confidential. The maintainer will review and respond. Possible
responses range from a private conversation to a temporary or permanent ban
from project spaces, depending on severity and pattern.
